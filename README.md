# 🐍 Snake Game (JavaScript Project)

## 📌 Overview

This is a browser-based Snake Game built using **HTML, CSS, and
JavaScript**. The project demonstrates how JavaScript can be used to
create a fully interactive game by handling real-time updates, user
input, and dynamic rendering.

------------------------------------------------------------------------

## 🚀 Features

-   Snake movement controlled using arrow keys
-   Real-time game loop using `setInterval`
-   Random food generation
-   Collision detection (walls)
-   Score tracking system
-   High score stored using `localStorage`
-   Timer to track gameplay duration
-   Start and Game Over modals
-   Restart functionality without page reload

------------------------------------------------------------------------

## 🧠 Role of JavaScript

JavaScript is the core of this project: - Controls snake movement and
direction - Handles keyboard input events - Updates the DOM
dynamically - Manages game states (start, running, game over) -
Implements collision detection logic - Generates food at random
positions - Tracks score and stores high score using `localStorage`

## ▶️ How to Run

1.  Download or clone the repository
2.  Open `index.html` in your browser
3.  Click **Start Game**
4.  Use arrow keys to control the snake

------------------------------------------------------------------------

## 🎮 Controls

-   ⬆️ Up Arrow → Move Up\
-   ⬇️ Down Arrow → Move Down\
-   ⬅️ Left Arrow → Move Left\
-   ➡️ Right Arrow → Move Right
